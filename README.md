# Diagpop

DiagFloat is a Neovim plugin that enhances your coding experience by
displaying diagnostic messages in customizable floating windows

## Installation

```lua
return {
  'maxmx03/diagpop.nvim',
  opts = {
    -- default config
    limit = 10,
    hl_group = 'NormalFloat',
    border = 'rounded',
  },
}
```
